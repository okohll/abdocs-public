Workflows

Using a view to create a workflow

A workflow can be used to edit data in the system automatically.

The view created in this case is used to define the set of records to act on (others can be excluded via view filters and also the data to use - each calculation or field in the view can be a source of data to create or update.

For more information, see the workflows topic, which contains full details of how to set up and use workflows.
